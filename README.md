# PianoKeys

## Description
PianoKeys is a web based piano that plays musical notes when you click the keys or press the corresponding keyboard keys. Each key has smooth animation effects to give a realistic piano experience.

## Features
- Play piano notes using mouse click
- Play piano notes using keyboard keys
- Smooth key press animation
- Responsive design for different screen sizes
